Computer Devices We Can Use to Operate the Windows System

Input Unit

The input unit allows users to enter data into the computer.

Examples: Keyboard, Mouse, Joystick, OMR, OCR

Keyboard

The keyboard is an input device that allows users to enter data into the computer.

There are three types of keys present on a standard 106-key keyboard:

  • Special Keys
  • Functional Keys
  • Typewriter Keys

1. Special Keys

Special keys include symbols and navigation buttons like Home, Enter, End, Page Up, and Page Down.

2. Functional Keys

Functional keys (F1-F12) are located at the top of the keyboard and serve different functions.

3. Typewriter Keys

The typewriter keys include alphabets (A-Z) and numbers (0-9).

Mouse

The mouse is another input device that allows users to interact with the computer.

Types of Mice:

1. Optical Mouse

2. Serial Mouse

3. PS/2 Mouse

4. Cordless Mouse

5. USB Mouse
